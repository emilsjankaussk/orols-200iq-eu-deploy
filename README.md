# orols.200iq.eu deploy bundle

Static deployment bundle for the OROLS demo site hosted on `orols.200iq.eu`.

Contents:

- `index.html`
- `assets/`
- `favicon.svg`
- `icons.svg`
- `.htaccess`
- `_redirects`

This repository is intended to be deployed directly by the Garmtech Git deployment panel without any server-side build step.
